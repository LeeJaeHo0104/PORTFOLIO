    1. 실시간 날씨 정보 구현
    async와 await를 사용하여 비동기적으로 날씨 정보를 표현하였습니다. 먼저 fetch 함수를 사용하여 OpenWeatherMap API로부터 날씨 데이터를 요청하였고,  이때, await를 사용하여 요청 결과가 도착할 때까지 기다립니다.
    응답에 성공하면, .json() 메서드를 사용하여 JSON 형태로 변환합니다. 이 과정도 비동기적으로 처리되므로 await를 사용합니다.
    변환된 JSON 데이터에서 필요한 정보(현재 온도, 최고 온도, 최저 온도, 체감 온도, 날씨 아이콘)를 추출하여 innerHTML을 이용해 요소에 삽입합니다.
    날씨 아이콘의 경우, 받아온 아이콘 코드를 사용하여 이미지 URL을 생성하고, 해당 이미지를 페이지에 표시합니다.  요청 과정에서 오류가 발생하면 catch 블록이 실행되어 에러 메시지를 출력합니다.

    2. 스크롤 애니메이션
    window.onscroll을 활용하여, 브라우저 창에서 스크롤 이벤트가 발생할 때마다 특정 함수가 실행되도록 설정하였습니다.  먼저 window.pageYOffset을 이용하여 스크롤된 높이 정보를 height 변수에 저장하였고, window.innerWidth를 사용하여 화면의 너비를 width 변수에 저장했습니다.  sec1Title이라는 변수를 대상으로, 화면의 너비가 1100px 이상인 경우와 그렇지 않은 경우를 구분하기 위해 if-else 조건문을 사용했습니다.
    if문 내에서는 화면 높이가 400px 이상인 경우와 그렇지 않은 경우를 구분했습니다.  else문 내에서는 화면 높이가 150px 이상인 경우와 그렇지 않은 경우를 구분했습니다.
    이렇게 설정함으로써, 다양한 화면 크기와 스크롤 위치에 따라 조건적으로 sec1Title에 다른 스타일을 적용할 수 있게 되었습니다.
    이러한 방식으로 반응형 디자인에서 설정한 조건에 따라 정상적으로 함수가 작동하도록 하였습니다.

    정해진 숫자까지 도달하면 clearInterVal 함수로 해당 동작을 중지시킵니다.

    2. JQuery를 활용한 FadeIN-Out 배너 구현
    slide 내부의 slide1, 2, 3 태그에 opacity : 0을 지정하고, on 클래스를 가진 태그만 opacity : 1을 가지게 하였다. slideFn함수내에서 removeClass를 이용하여 on 클래스를 제거하고,
    addClass를 이용하여 on 클래스를 넣어줍니다.  addClass의 대상이 되는 변수는 해당 함수가 실행됨에 따라 idx 변수의 값이 1씩 증가해 해당 index 값을 가진 변수가 할당됩니다.
    또한, idx 변수가 slide의 length와 같아지면 다시 idx의 값을 0으로 만들어 재실행 하게 하였습니다. slideFn 함수는 setInterVal 함수를 이용해 3초에 한번씩 실행하게 합니다.
    이러한 동작으로 3초에 한번씩 배너 이미지가 변경되는 동작이 실행됩니다.
