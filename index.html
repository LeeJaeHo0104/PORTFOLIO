    Description (Project1)
    1. 현재 시간 나타내기
    time 함수를 구현하여 new Date 함수를 통해 현재 시간을 가져오고, toLocaleTimeString 함수를 활용하여 현재 시간을 'HH:MM:SS' 형식으로 변환하였습니다.  이후, innerHTML 속성을 사용하여 id가 'time'인 HTML 요소에 해당 시간을 표시하도록 하였습니다.  또한, setInterval 함수를 이용해 time 함수가 매 1초(1000ms)마다 반복 실행되어 시간 정보가 지속적으로 업데이트 되도록 설정하였습니다.
    이러한 코드를 구현하여 사용자는 웹 페이지 상에서 실시간으로 갱신되는 정확한 시간을 확인할 수 있도록 하는 기능을 만들었습니다.

    2. 슬라이드 배너
    timer 함수를 통해 일정 시간마다 slideFn 함수를 호출하여 배너가 자동으로 넘어가도록 설정하고, 배너 위에 마우스를 올리면 clearInterval 함수를 이용해 중지되도록 합니다.  사용자가 'next' 버튼을 클릭하면 배너가 left를 기준으로 0%에서 -100%로 이동하고, 'prev' 버튼을 클릭하면 배너의 left를 기준으로 0%에서 100%로 이동하여 새로운 배너가 나타나는 애니메이션 효과가 실행됩니다.  페이저 버튼을 클릭하면 해당 배너로 직접 이동할 수 있습니다.  모든 애니메이션은 move 함수를 통해 구현되며, 이 함수는 시작 위치와 끝 위치를 매개변수로 받아 CSS 속성을 변경하고 애니메이션을 적용합니다.

    1. Gnb 스티키 메뉴
    cloneNode 함수와 true 인자를 사용하여 원래의 gnb 메뉴와 그 자식 요소들을 모두 복제했습니다.  이를 통해 원본 gnb 메뉴의 복사본을 변수 gnbClonedp 저장하였습니다.  이후, appendChild 함수를 이용해 복제된 gnb 메뉴를 gnbStickyContainer 변수에 추가하였고, 이 과정을 통해 복제된 메뉴가 변수 gnbStickyContainer 안에 위치하게 되었습니다.  마지막으로 setAttribute 함수를 사용해 복제된 gnb 메뉴 안에 있는 로고 이미지의 소스를 기존의 것과 다른 빨간색 로고 이미지로 변경하였습니다.

    2. 슬라이드 배너
    window.onscroll을 활용하여, 브라우저 창에서 스크롤 이벤트가 발생할 때마다 특정 함수가 실행되도록 설정하였습니다.  먼저 window.pageYOffset을 이용하여 스크롤된 높이 정보를 height 변수에 저장하였고, window.innerWidth를 사용하여 화면의 너비를 width 변수에 저장했습니다.  sec1Title이라는 변수를 대상으로, 화면의 너비가 1100px 이상인 경우와 그렇지 않은 경우를 구분하기 위해 if-else 조건문을 사용했습니다.  if문 내에서는 화면 높이가 400px 이상인 경우와 그렇지 않은 경우를 구분했습니다.  else문 내에서는 화면 높이가 150px 이상인 경우와 그렇지 않은 경우를 구분했습니다.  이렇게 설정함으로써, 다양한 화면 크기와 스크롤 위치에 따라 조건적으로 sec1Title에 다른 스타일을 적용할 수 있게 되었습니다. 이러한 방식으로 반응형 디자인에서 설정한 조건에 따라 정상적으로 함수가 작동하도록 하였습니다.
